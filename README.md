# datacheker
ギガペースチェッカー

概要

ギガペースチェッカーは、月間データ通信量の消費ペースを確認するためのシンプルなWebツールです。
現在使用している通信量を入力すると、月末までにギガが足りるかどうかを自動計算し、通信量の使用ペースを判定します。

このツールでは次の情報を表示します。

- ギガの消費ペース（余裕 / 適正 / 使いすぎ）
- 月末予測使用量
- 今日使える通信量
- データ通信節約Tips

Androidスマートフォンのブラウザから利用できる軽量なツールとして設計されています。

---

主な機能

1. 通信量ペース判定

現在の通信量と月の進行日数を比較し、ギガの使用ペースを判定します。

判定結果は次の3段階です。

- 余裕
- 適正
- 使いすぎ

---

2. 月末予測使用量

現在の使用ペースから、月末までにどれくらいの通信量を使うかを予測します。

---

3. 今日使える通信量

月末までにギガを使い切らないために、
「今日あとどれくらい使えるか」を計算して表示します。

---

4. 節約Tips表示

通信量を節約するための簡単なTipsをランダムで表示します。

例

- 動画の画質を下げる
- アプリ更新をWi-Fiのみにする
- 自動動画再生をOFFにする

---

使い方

1. 月間通信容量を選択（例：1GB / 3GB / 5GB）
2. 現在使用している通信量を入力
3. ペース判定・予測使用量・今日使える通信量を確認

※データリセット日は 毎月1日 を前提としています。

---

計算ロジック

日数進行率

今日の日付 ÷ 月の日数

使用率

現在使用量 ÷ 月間通信容量

ペース判定

差 = 使用率 − 日数進行率

条件| 判定
-5% 未満| 余裕
-5%〜+5%| 適正
+5% 超| 使いすぎ

---

月末予測使用量

現在使用量 ÷ 経過日数 × 月日数

---

今日使える通信量

(月容量 − 使用量) ÷ 残日数
